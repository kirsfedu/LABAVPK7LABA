# Веб-сервис регистрации пользователей

## Название и назначение
**User Registration API** — веб-сервис для регистрации, хранения и управления пользовательскими данными.

## Решаемая задача и целевая аудитория
**Задача:** Создание безопасного и масштабируемого API для управления пользователями в веб-приложениях.  
**Целевая аудитория:** 
- Разработчики, интегрирующие аутентификацию в свои приложения
- Студенты, изучающие ASP.NET Core и REST API
- Администраторы систем, управляющие пользователями

## Основные функции
1. **Регистрация пользователя**  
   Принимает логин, email и пароль, хэширует пароль и сохраняет в памяти.
2. **Получение списка всех пользователей**  
   Возвращает JSON-массив всех зарегистрированных пользователей.
3. **Получение пользователя по ID**  
   Возвращает данные конкретного пользователя.

## Архитектурные особенности
- **Язык и фреймворк:** C#, ASP.NET Core 8
- **Библиотеки:** Swashbuckle.AspNetCore (Swagger)
- **Хранение данных:** In-memory коллекция (можно заменить на БД)
- **Аутентификация:** Пока не реализована (планируется JWT)
- **Документация:** Swagger UI с XML-комментариями

## Инструкции по развертыванию и запуску
1. Клонировать репозиторий
2. Открыть проект в Visual Studio 2022 или выше
3. Восстановить NuGet-пакеты
4. Запустить проект (F5)
5. Перейти по ссылке: `https://localhost:5043/swagger`

Для работы требуется:
- .NET 8 SDK
- IDE (Visual Studio / VS Code / Rider)
- Браузер для тестирования
